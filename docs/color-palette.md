# Color Palette

## Brand

- `#081217` `Ink`: primary surface and headline color
- `#14B8A6` `Mint`: main brand accent
- `#38BDF8` `Sky`: secondary accent for gradients and highlights
- `#5EEAD4` `Soft Mint`: light accent for badges and hover states

## Neutral

- `#FFFFFF` `White`: card surface
- `#F4F7F9` `Cloud`: page background
- `#CBD5E1` `Line`: border and divider
- `#475569` `Slate`: secondary text

## Feedback

- `#DCFCE7` `Success Surface`
- `#166534` `Success Text`
- `#FEF3C7` `Warning Surface`
- `#92400E` `Warning Text`
- `#E2E8F0` `Disabled Surface`
- `#94A3B8` `Disabled Text`

## Usage Notes

- Use `Ink` for hero areas, strong buttons, and navigation emphasis.
- Use `Mint` and `Sky` together for gradients, charts, and focus rings.
- Use `Cloud` as the default layout background to keep cards readable.
- Reserve feedback colors for state communication, not primary branding.
