# Color Palette

## Brand

- `#185FA5` `Primary`: 메인 브랜드 컬러 (버튼, 강조, 링크)
- `#0C447C` `Primary Dark`: 헤더 텍스트, 강한 강조
- `#E6F1FB` `Primary Light`: 배경 틴트, 뱃지, 호버 상태

## Neutral

- `#FFFFFF` `White`: 카드 표면
- `#F4F7F9` `Cloud`: 페이지 배경
- `#E5E7EB` `Line`: 테두리, 구분선 (gray-200)
- `#6B7280` `Slate`: 보조 텍스트 (gray-500)
- `#9CA3AF` `Muted`: 힌트 텍스트 (gray-400)

## Feedback

- `#DCFCE7` / `#166534` `Success` (surface / text)
- `#FEF3C7` / `#92400E` `Warning` (surface / text)
- `#FEE2E2` / `#DC2626` `Error` (surface / text)
- `#E2E8F0` / `#94A3B8` `Disabled` (surface / text)

## Usage Notes

- `Primary` (#185FA5): 주요 버튼, 링크, 선택 상태, 프로그레스 바
- `Primary Dark` (#0C447C): 페이지 타이틀, 강한 강조 텍스트
- `Primary Light` (#E6F1FB): 대시보드 배경 틴트, 선택된 카드 배경
- `Cloud` (#F4F7F9): 전체 레이아웃 배경
- 텍스트 최소 대비: gray-500 이상 사용 권장 (WCAG AA)
- 힌트/서브레이블은 gray-400 — 배경 흰색 시 대비 주의
